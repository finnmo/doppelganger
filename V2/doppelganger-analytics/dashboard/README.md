# Instagram Analytics Dashboard

Interactive Next.js dashboard for visualizing Instagram message analytics generated by Doppelgänger Analytics.

## 🚀 Quick Start

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Copy Data Files**
   ```bash
   mkdir -p public/data
   cp ../dash-data/*.json public/data/
   ```

3. **Start Development Server**
   ```bash
   npm run dev
   ```

4. **Open Dashboard**
   Navigate to `http://localhost:3000`

## 🔧 Development

### **Available Scripts**
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

### **Adding New Charts**
1. Create component in `src/components/`
2. Add data fetching logic with conversation filtering
3. Include in appropriate tab
4. Add TypeScript interfaces for data

### **Styling Guidelines**
- Use Tailwind utility classes
- Follow existing color schemes
- Implement responsive design patterns
- Ensure accessibility with proper ARIA labels

|------|-------------|
| `textMetrics.json` | Basic message statistics |
| `sentimentBySender.json` | Sentiment analysis by participant |
| `emotionMetrics.json` | Emotion detection results |
| `reactionMetrics.json` | Reaction analysis |
| `mediaMetrics.json` | Media sharing patterns |
| `conversationMetrics.json` | Conversation-level analytics |
| And 15+ additional specialized data files |
